# 13. Required Deployment Files

Every deployment must include the full application stack.

Backend

• API routes
• services
• middleware
• utilities

Frontend

• frontend source files
• compiled production build

Configuration

• next.config
• tailwind.config
• postcss.config
• environment templates
• build configuration files

Assets

• public directory
• images
• fonts
• static assets

Database Layer

• Prisma schema
• migrations
• Prisma client
